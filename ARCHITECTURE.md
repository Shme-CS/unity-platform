# Architecture Documentation

## Overview

Unity Platform is built with a modern React architecture using TypeScript and Tailwind CSS.

## Component Structure

### UI Components
- Button: Reusable button with variants
- Card: Container with hover effects
- Badge: Small label component
- Container: Responsive wrapper
- SectionTitle: Consistent headers

### Section Components
- Hero: Landing section
- Stats: Key metrics
- Features: Platform features
- Courses: Course listings
- And 9 more sections...

### Layout Components
- Navbar: Navigation bar
- Footer: Site footer

## State Management

Currently using React hooks. Ready for Redux/Zustand integration.

## Styling

Tailwind CSS utility-first approach with custom theme configuration.

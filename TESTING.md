# Testing Guide

## Manual Testing

### Development Server
```bash
npm run dev
```

### Production Build
```bash
npm run build
npm run preview
```

## Browser Testing

Test on:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Responsive Testing

Test viewports:
- Mobile: 375px
- Tablet: 768px
- Desktop: 1440px

## Accessibility Testing

- Keyboard navigation
- Screen reader compatibility
- Color contrast
- Focus states
